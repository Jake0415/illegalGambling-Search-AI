## 개요

불법 도박 사이트 자동 채증 시스템의 2단계(회원가입) 채증에서 **SMS OTP 인증**은 최대 장벽이다. 실제 테스트 결과(2026-03-14, Wego88 등), 대부분의 불법 도박 사이트는 CAPTCHA 없이 SMS OTP만 요구하며, PASS/아이핀/카카오/네이버 등 정식 인증 수단은 불법 사이트가 정식 계약 불가하여 사용하지 않는다.

본 섹션은 SMS 가상번호 발급부터 OTP 수신, 인증코드 입력, 실패 시 폴백 및 수동 개입까지 전체 인증 자동화 플로우의 기능 요구사항을 정의한다.

### 핵심 의사결정 (decisions.md 반영)

| 결정 | 내용 |
|------|------|
| 1순위 SMS 서비스 | PVAPins Non-VoIP (실제 통신사 SIM 기반) |
| 2순위 폴백 | GrizzlySMS / SMS-Activate (VoIP) |
| 3순위 폴백 | 수사용 실제 선불 SIM 카드 풀 |
| 최종 폴백 | Slack 알림 → 담당자 수동 인증 |

## 인증 플로우 다이어그램

```
사이트 접속 → 회원가입 폼 탐지 → 전화번호 필드 식별
    → PVAPins 번호 발급 → 폼에 입력 → OTP 요청 클릭
    → SMS 수신 대기 (최대 120초)
    → [성공] OTP 파싱 → 인증코드 입력 → 다음 단계
    → [실패] 폴백 번호 발급 (GrizzlySMS) → 재시도
    → [2차 실패] SMS-Activate 폴백 → 재시도
    → [전체 실패 3회] Slack 알림 → 수동 개입 큐
```

```
┌─────────────────────────────────────────────────┐
│              SMS 인증 자동화 엔진                  │
│                                                  │
│  1. 전화번호 필드 탐지 (Claude Vision 연계)        │
│  2. SMS 서비스에서 +82 번호 발급                   │
│  3. 폼에 번호 입력 → OTP 요청 버튼 클릭            │
│  4. SMS API 폴링으로 OTP 코드 수신 (최대 120초)    │
│  5. OTP 코드 파싱 → 인증코드 필드에 입력            │
│  6. 인증 완료 → 다음 채증 단계로 진행               │
│                                                  │
│  ┌──────────┐  실패  ┌──────────┐  실패  ┌──────────┐ │
│  │ PVAPins  │──────→│GrizzlySMS│──────→│SMS-      │ │
│  │(Non-VoIP)│       │ (VoIP)   │       │Activate  │ │
│  └──────────┘       └──────────┘       └──────────┘ │
│        │                 │                 │        │
│        └───── 모두 실패 ──┴──── 수동 개입 ──┘        │
└─────────────────────────────────────────────────┘
```

## 법적 고려사항

본 SMS 인증 자동화 기능은 불법 도박 사이트 채증을 위한 수사 목적으로 사용된다. 다음 법적 사항을 준수해야 한다:

1. **가상번호 사용 적법성**: 수사기관 법무팀의 사전 법적 검토 완료 후 운영 (decisions.md 반영)
2. **증거 보전**: SMS 인증 과정 전체를 Chain of Custody에 포함하여 증거 무결성 확보
3. **개인정보 보호**: 가상번호 및 OTP 데이터는 수사 목적으로만 사용, 최소 보존 원칙 적용
4. **위장수사 범위**: 기회제공형 수사 범위 내에서 운영 (이미 운영 중인 불법 사이트 대상)
5. **관련 법률**: 전기통신사업법 제83조, 통신비밀보호법 제13조, 국민체육진흥법

> **주의**: 가상번호를 이용한 불법 도박사이트 위장가입에 대한 직접적 판례는 확인되지 않았으므로, 운영 전 반드시 수사기관 법무팀의 적법성 확인을 권장한다.
