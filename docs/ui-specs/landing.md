# 랜딩 페이지

## 페이지 목적

비인증 사용자에게 시스템의 핵심 가치(불법 도박 사이트 자동 탐지/채증/증거 확보)를 소개하고, 로그인 또는 데모 요청으로 유도한다. 인증된 사용자는 대시보드로 자동 리디렉트된다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/` (비인증 상태) | 공개 랜딩 페이지 |

#### 컴포넌트

| 컴포넌트 | 출처 | 용도 |
|----------|------|------|
| `Button` | shadcn/ui | CTA (로그인, 데모 요청) |
| `Card` | shadcn/ui | 기능 카드 6개, 기관 카드 4개 |
| Lucide Icons | lucide-react | `Search`, `Camera`, `Brain`, `ShieldCheck`, `Phone`, `FileText`, `Building2`, `Shield`, `Landmark` |
| `next/image` | Next.js | 대시보드 목업 이미지 |
| 커스텀 `PipelineDiagram` | 커스텀 SVG/CSS | 시스템 구성도 플로우 다이어그램 |
| 커스텀 `LandingNav` | 커스텀 | 고정 네비게이션 바 |
| 커스텀 `LandingFooter` | 커스텀 | 푸터 (연락처, 법적 고지, 저작권) |

| 항목 | 값 |

#### 인터랙션

| 동작 | 반응 |
|------|------|
| "로그인" CTA 클릭 | `/login`으로 이동 |
| "데모 요청" CTA 클릭 | `mailto:admin@gambling-watch.kr` 또는 외부 폼 링크 |
| 기능 카드 호버 | 스케일 업 + 그림자 확대 애니메이션 |
| 네비게이션 링크 클릭 | 해당 섹션으로 스무스 스크롤 |
| 인증된 사용자 접근 | `/` → 자동 리디렉트 → 대시보드 |
| 다크 모드 | 배경, 카드, 텍스트, CTA 색상 전환 |
| 모바일 (375px) | 네비게이션 햄버거 메뉴, 카드 1열, 푸터 수직 정렬 |
| 태블릿 (768px) | 카드 2열, 기관 2열 |
| 데스크톱 (1920px) | 카드 3열, 기관 4열 |
