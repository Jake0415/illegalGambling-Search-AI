# Task 011: 메인 대시보드

## 페이지 목적

시스템 핵심 KPI(탐지 사이트 수, 채증 성공률, 진행 중 작업, 무결성 검증률)를 한눈에 파악하고, 최근 시스템 활동 피드를 실시간으로 확인하며, 현재 진행 중인 채증 작업의 실시간 상태와 외부 서비스 운영 상태를 모니터링한다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/` (인증 상태) | 메인 대시보드 (KPI, 활동 피드, 채증 모니터링, 시스템 상태) |

#### 컴포넌트

| 컴포넌트 | 출처 | 용도 |
|----------|------|------|
| `PageContainer` | 커스텀 공통 | 페이지 헤더 |
| `Card`, `CardHeader`, `CardContent` | shadcn/ui | KPI 카드, 채증 진행 카드, 시스템 상태 카드 |
| `CardSkeleton` | 커스텀 공통 | KPI 로딩 스켈레톤 |
| `Progress` | shadcn/ui | 채증 진행률 바 |
| `Badge` | shadcn/ui | 활동 유형 배지, 상태 인디케이터 |
| `StatusBadge` | 커스텀 공통 | 채증 상태, 서비스 상태 |
| `ScrollArea` | shadcn/ui | 활동 피드 스크롤 |
| `Button` | shadcn/ui | 더 보기, 수동 새로고침 |
| `ToggleGroup` | shadcn/ui | 활동 피드 유형 필터 |
| `Tooltip` | shadcn/ui | KPI 증감률 상세, 서비스 상세 |
| `Alert` | shadcn/ui | 외부 서비스 경고/오류 배너 |
| Lucide Icons | lucide-react | `TrendingUp`/`TrendingDown` (KPI 증감), 활동 유형 아이콘 |

#### 데이터

| 항목 | 값 |
|------|-----|
| Mock 서비스 (KPI) | `mockAnalyticsService.getOverview()` |
| API 엔드포인트 | `GET /api/dashboard/summary` |
| 응답 타입 | `ApiResponse<DashboardSummaryData>` |

**DashboardSummaryData 필드 매핑:**

| KPI 카드 | 필드 | 부가 정보 |
|----------|------|-----------|
| 총 탐지 사이트 | `totalSites` (30) | `newSites.today` (2) — 금일 신규 |
| 채증 성공률 | `successRates.stage3` (87%) | 목표선 70% 대비 표시 |
| 진행 중 작업 | `investigations.in_progress` (3) | `investigations.queued` (5) — 대기 건수 |
| 무결성 검증률 | 계산값 (100%) | 전체 증거 대비 VALID 비율 |

**시스템 상태 패널:**

| 항목 | 필드 | 상태 기준 |
|------|------|-----------|
| SMS | `externalServices.smsProvider` | healthy=green, degraded=yellow, down=red |
| CAPTCHA | `externalServices.captchaSolver` | 동일 |
| 프록시 | `externalServices.proxy` | 동일 |
| 시스템 가동률 | `system.uptime` | >=99%=green, >=95%=yellow, <95%=red |
| 수동 개입 큐 | `manualQueue.pending`, `avgWaitTime` | pending>0이면 orange |

**활동 피드:**

| 항목 | 값 |
|------|-----|
| API 엔드포인트 | `GET /api/dashboard/activities` |
| 응답 타입 | `ActivityItem[]` |
| 실시간 업데이트 | SSE (`/api/dashboard/stream`) |
| 초기 로드 | 최근 50건 |
| 추가 로드 | `useInfiniteQuery` 무한 스크롤 |

#### 인터랙션

| 동작 | 반응 |
|------|------|
| KPI "총 탐지 사이트" 카드 클릭 | `/sites`로 이동 |
| KPI "채증 성공률" 카드 클릭 | `/investigations`로 이동 |
| KPI "진행 중 작업" 카드 클릭 | `/investigations?status=IN_PROGRESS`로 이동 |
| KPI "무결성 검증률" 카드 클릭 | `/evidence`로 이동 |
| 채증 진행 카드 클릭 | `/investigations/[id]`로 이동 |
| SSE 이벤트 수신 | 채증 진행률 실시간 업데이트 (0.5~1초 간격) |
| 활동 피드 유형 필터 클릭 | 해당 유형만 표시 |
| 활동 피드 항목 클릭 | 관련 상세 페이지로 이동 (사이트/채증/증거) |
| "더 보기" 클릭 | 다음 50건 로드 |
| 시스템 상태 "수동 새로고침" 클릭 | 즉시 상태 재조회 |
| 서비스 경고/오류 상태 | 대시보드 상단에 Alert 배너 표시 |
| 서비스 잔액 20% 이하 | 해당 서비스 yellow 경고 표시 |
