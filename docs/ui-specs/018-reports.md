# Task 018: 보고서 페이지

## 페이지 목적

월간/주간 정기 보고서를 자동 생성하고 관리하며, 보고서 및 통계 데이터를 PDF/Excel 형식으로 내보내어 기관별 양식에 맞는 보고서를 제출한다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/reports` | 보고서 목록 및 생성/내보내기 |

---

#### 컴포넌트

| 컴포넌트 | 출처 | 용도 |
|----------|------|------|
| `PageContainer` | 커스텀 공통 | 페이지 헤더 |
| `Card` | shadcn/ui | 스케줄 카드, 내보내기 섹션 |
| `DataTable` | 커스텀 공통 | 보고서 목록 테이블 |
| `Badge` | shadcn/ui | 유형 (주간/월간/커스텀), 상태 (완료/생성 중/실패) |
| `Progress` | shadcn/ui | 생성 중 진행률 |
| `Select` | shadcn/ui | 유형 필터 |
| `DatePicker` | shadcn/ui | 기간 필터 |
| `RadioGroup` | shadcn/ui | 내보내기 형식 선택 |
| `Checkbox` | shadcn/ui | 기관 양식, 포함 데이터 선택 |
| `Switch` | shadcn/ui | 스케줄 활성/비활성 |
| `Button` | shadcn/ui | 보기, 다운로드, 내보내기 실행 |
| `Dialog` | shadcn/ui | 보고서 미리보기, 스케줄 편집 |
| `toast` (Sonner) | shadcn/ui | 생성/다운로드 결과 |

#### 데이터

| 항목 | 값 |
|------|-----|
| API 엔드포인트 (목록) | `GET /api/reports` |
| API 엔드포인트 (생성) | `POST /api/reports` |
| API 엔드포인트 (내보내기) | `POST /api/reports/export` |
| 보고서 포함 내용 | 탐지 통계, 채증 통계, 도메인 호핑, 주요 사이트, 비용, AI 인사이트 |

#### 인터랙션

| 동작 | 반응 |
|------|------|
| "보고서 생성" 클릭 | Dialog: 유형/기간/포함항목 선택 → 생성 시작 → 목록에 "생성 중" 상태 추가 |
| "보기" 클릭 | PDF 미리보기 Dialog |
| "다운로드" 클릭 | PDF/XLSX 파일 다운로드 |
| "내보내기 실행" 클릭 | 선택된 형식/양식/데이터로 파일 생성 + 다운로드 |
| 스케줄 "편집" 클릭 | 크론 표현식/시간/발송 채널 편집 Dialog |
| 생성 완료 이벤트 (SSE) | 상태를 "완료"로 업데이트 + toast |
