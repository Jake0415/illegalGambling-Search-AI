# Task 019: 설정 페이지

## 페이지 목적

시스템 전반의 운영 설정(외부 서비스 API 키, 프록시, 채증 파이프라인 등)을 관리하고, 사용자 계정과 역할 기반 권한을 관리하며, 불법 도박 탐지용 키워드를 CRUD 관리하고, 시스템 전체 감사 로그를 조회하여 운영 투명성을 확보한다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/settings` | 시스템 설정 (탭 기반 설정 폼) |
| `/settings/users` | 사용자/역할 관리 |
| `/settings/keywords` | 키워드 관리 |
| `/settings/audit-log` | 감사 로그 뷰어 (시스템 전체) |
