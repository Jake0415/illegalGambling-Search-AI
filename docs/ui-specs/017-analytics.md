# Task 017: 통계 및 분석 페이지

## 페이지 목적

불법 도박 사이트 탐지 및 채증 현황의 기간별 추이를 차트로 시각화하고, 카테고리별 사이트 분포를 파이/바 차트로 확인하며, 도메인 호핑 관계를 인터랙티브 네트워크 그래프로 시각화하여 운영 인사이트를 도출한다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/analytics` | 통계 대시보드 (추이 차트 + 카테고리 분포 + 네트워크 그래프) |

#### 컴포넌트

| 컴포넌트 | 출처 | 용도 |
|----------|------|------|
| `PageContainer` | 커스텀 공통 | 페이지 헤더 |
| `Card` | shadcn/ui | 각 섹션 래퍼 |
| `ToggleGroup` | shadcn/ui | 기간 선택 (7일/30일/90일/1년) |
| `DatePicker` | shadcn/ui | 커스텀 기간 |
| Recharts `LineChart` | recharts | 탐지 추이 (일별 탐지 수, 출처별) |
| Recharts `AreaChart` | recharts | 채증 성공/실패 스택 영역 |
| Recharts `PieChart` | recharts | 카테고리 파이, 상태 도넛 |
| Recharts `BarChart` | recharts | 카테고리 수평 바 |
| Recharts `Tooltip`, `Legend` | recharts | 호버 상세, 범례 |
| D3.js / Cytoscape.js | 외부 라이브러리 | 네트워크 그래프 (Phase 3) |
| `Button` | shadcn/ui | CSV 내보내기, 줌 컨트롤 |
| `Slider` | shadcn/ui | 시간 범위 슬라이더 (네트워크) |
| `Sheet` | shadcn/ui | 노드 클릭 시 사이드 패널 |
| `CardSkeleton` | 커스텀 공통 | 차트 로딩 |

#### 데이터

| 항목 | 값 |
|------|-----|
| Mock 서비스 (탐지 통계) | `mockAnalyticsService.getSiteStats(filter: StatsFilter)` |
| Mock 서비스 (채증 통계) | `mockAnalyticsService.getInvestigationStats(filter: StatsFilter)` |
| Mock 서비스 (카테고리) | `mockAnalyticsService.getCategoryDistribution()` |
| 탐지 응답 타입 | `DetectionStatsData { period, data[] { date, detectedCount, bySource, byCategory, newVsExisting, precision } }` |
| 채증 응답 타입 | `InvestigationStatsData { period, data[] { date, totalInvestigations, successRateByStage, avgDuration, failureReasons, manualInterventionRate } }` |
| 카테고리 응답 타입 | `CategoryDistributionData { categories[] { category, count, percentage, trend }, unclassifiedCount }` |
| 필터 | `StatsFilter { period?, from?, to? }` |

**차트별 데이터 매핑:**

| 차트 | 데이터 소스 | X축 | Y축/시리즈 |
|------|------------|-----|-----------|
| 탐지 추이 | `DetectionStatsData.data` | `date` | `detectedCount`, `bySource.*` |
| 채증 성공/실패 | `InvestigationStatsData.data` | `date` | `totalInvestigations` (성공/실패 분리) |
| 성공률 추이 | `InvestigationStatsData.data` | `date` | `successRateByStage.stage3` + 70% 목표선 |
| 카테고리 파이 | `CategoryDistributionData.categories` | - | `category`, `count`, `percentage` |
| 카테고리 바 | `CategoryDistributionData.categories` | `category` | `count` |

#### 인터랙션

| 동작 | 반응 |
|------|------|
| 기간 선택 변경 | 모든 차트 데이터 재조회 + 재렌더링 |
| 차트 호버 | Recharts Tooltip으로 해당 날짜 상세 수치 |
| 파이/바 차트 세그먼트 클릭 | `/sites?category=해당카테고리`로 이동 |
| 범례 클릭 | 해당 시리즈 토글 (표시/숨기기) |
| "CSV 내보내기" 클릭 | 현재 기간 통계 데이터를 CSV 다운로드 (papaparse) |
| 네트워크 노드 드래그 | 노드 위치 변경 |
| 네트워크 노드 클릭 | 우측 Sheet 패널에 사이트 정보 표시 |
| 네트워크 줌 | 마우스 휠 또는 버튼으로 줌 인/아웃 |
| 시간 범위 슬라이더 | 네트워크 그래프에서 특정 기간의 연결 관계만 표시 |
