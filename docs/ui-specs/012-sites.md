# Task 012: 사이트 관리 페이지

## 페이지 목적

탐지된 불법 도박 사이트의 전체 목록을 조회하고, 신규 URL을 수동/벌크로 등록하며, 개별 사이트의 상세 정보(WHOIS, DNS, 채증 이력, 증거 파일 등)를 종합적으로 확인하여 상태를 관리한다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/sites` | 사이트 목록 (필터, 정렬, 페이지네이션) |
| `/sites/[id]` | 사이트 상세 정보 (탭 기반) |
| `/sites/new` | 수동 URL 등록 + 채증 시작 |
| `/sites/import` | 벌크 CSV/엑셀 임포트 |
