# Task 014: 증거 관리 페이지

## 페이지 목적

생성된 증거 패키지(스크린샷, HTML, WARC, 네트워크 로그 등)의 전체 목록을 조회하고 다운로드하며, SHA-256 해시 및 타임스탬프(OTS/RFC 3161) 기반 무결성 검증을 수행하고, 법원 제출용 보고서를 생성하며, 증거 체인 오브 커스터디(Chain of Custody) 감사 로그를 추적한다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/evidence` | 증거 패키지 목록 (테이블, 다운로드) |
| `/evidence/[id]/verify` | 증거 무결성 검증 UI |
| `/evidence/[id]/report` | 증거 보고서 생성 및 미리보기 |
| `/evidence/[id]/audit` | 증거 체인 오브 커스터디 감사 로그 |
