# Task 015: CAPTCHA/OTP 수동 개입

## 페이지 목적

자동 CAPTCHA/OTP 풀이가 실패하여 수동 개입이 필요한 세션의 대기 큐를 관리하고, CDP WebSocket 기반 원격 브라우저 뷰어를 통해 실시간으로 CAPTCHA/OTP를 수동 풀이하며, 사용자별 알림 채널(Slack/웹 푸시)을 설정한다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/investigations/captcha-queue` | CAPTCHA/OTP 수동 개입 큐 (카드 목록) |
| (모달) 원격 브라우저 뷰어 | CDP WebSocket 실시간 브라우저 스트리밍 |
| (설정 내) 알림 설정 | Slack/웹 푸시 알림 구성 |

> 참고: `/investigations/captcha-queue` 큐 목록의 기본 와이어프레임은 `013-investigations.md`에 정의되어 있다. 이 문서는 원격 브라우저 뷰어(FR-UI-020)와 알림 설정(FR-UI-021)을 추가로 정의한다.

### 원격 브라우저 뷰어 (모달/패널)

#### 레이아웃

```
+------------------------------------------------------------------+
| Dialog (fullscreen 또는 대형 모달)                                |
| title="원격 브라우저 - lucky-baccarat.asia"                       |
+------------------------------------------------------------------+
|                                                                    |
| +-- 상태 바 ---------------------------------------------------+ |
| | 연결: [v 연결됨]  남은 시간: 4:32  세션: cdp-session-abc123   | |
| +---------------------------------------------------------------+ |
|                                                                    |
| +-- 브라우저 화면 스트리밍 (Canvas/Img) -----------------------+ |
| |                                                                | |
| |  +----------------------------------------------------------+ | |
| |  |                                                          | | |
| |  |           [원격 브라우저 실시간 화면]                     | | |
| |  |                                                          | | |
| |  |     마우스 클릭, 키보드 입력이 원격 브라우저에 전달       | | |
| |  |                                                          | | |
| |  |     reCAPTCHA v2 체크박스가 표시된 상태                   | | |
| |  |     ┌──────────────────┐                                 | | |
| |  |     │ [ ] 로봇이 아닙니다 │                              | | |
| |  |     └──────────────────┘                                 | | |
| |  |                                                          | | |
| |  +----------------------------------------------------------+ | |
| |                                                                | |
| | 해상도: 1280x720  |  FPS: 8  |  지연: 120ms                  | |
| +---------------------------------------------------------------+ |
|                                                                    |
| +-- 액션 바 ---------------------------------------------------+ |
| | [풀이 완료] [타임아웃 연장 +5분] [세션 종료] [전체 화면]      | |
| +---------------------------------------------------------------+ |
+------------------------------------------------------------------+
```

#### 컴포넌트

| 컴포넌트 | 출처 | 용도 |
|----------|------|------|
| `Dialog` | shadcn/ui | 뷰어 모달 (lg/fullscreen) |
| HTML5 `<canvas>` | 네이티브 | CDP screencastFrame 렌더링 |
| `Badge` | shadcn/ui | 연결 상태 (연결됨=green, 연결 중=yellow, 끊김=red) |
| `Button` | shadcn/ui | 풀이 완료, 연장, 종료, 전체 화면 |
| `Progress` | shadcn/ui | 타임아웃 카운트다운 바 |
| `Alert` | shadcn/ui | 연결 끊김 경고 |
| `toast` (Sonner) | shadcn/ui | 풀이 완료, 연결 재시도 알림 |

#### 데이터

| 항목 | 값 |
|------|-----|
| API 엔드포인트 | `POST /api/manual-queue/:id/stream` |
| 응답 타입 | `CdpStreamData { wsUrl, sessionId, expiresAt, browserInfo }` |
| WebSocket | CDP `Page.startScreencast` → `Page.screencastFrame` (5~10 fps) |
| 입력 전달 | `Input.dispatchMouseEvent`, `Input.dispatchKeyEvent` |

#### 인터랙션

| 동작 | 반응 |
|------|------|
| 큐 카드 "풀이 시작" 클릭 | CDP WebSocket 연결 → 뷰어 모달 열림 → 화면 스트리밍 시작 |
| 캔버스 위 마우스 클릭 | `Input.dispatchMouseEvent` (좌표 변환하여 전달) |
| 캔버스 위 마우스 이동 | `Input.dispatchMouseEvent` (mouseMoved) |
| 키보드 입력 | `Input.dispatchKeyEvent` (keyDown, keyUp, char) |
| "풀이 완료" 클릭 | 세션 해결 API → 채증 자동화 재개 → 뷰어 닫힘 |
| "타임아웃 연장" 클릭 | +5분 연장 → 카운트다운 갱신 |
| "세션 종료" 클릭 | ConfirmDialog → 세션 종료 + 뷰어 닫힘 |
| CAPTCHA 풀이 자동 감지 (URL/DOM 변경) | toast("CAPTCHA 풀이가 감지되었습니다") → 채증 자동 재개 → 뷰어 닫힘 |
| WebSocket 연결 끊김 | 자동 재연결 3회 시도 → 실패 시 Alert 표시 |

### 알림 설정 (설정 내 패널)

#### 레이아웃

```
+------------------------------------------------------------------+
| Card                                                              |
| title="알림 설정"                                                 |
+------------------------------------------------------------------+
|                                                                    |
| +-- 알림 채널 -------------------------------------------------+ |
| |                                                                | |
| | Slack 알림                                                     | |
| | [v 활성화]                                                    | |
| | Webhook URL: [https://hooks.slack.com/...____] [테스트 전송]  | |
| |                                                                | |
| | 웹 푸시 알림                                                   | |
| | [v 활성화]  [구독 등록] (또는 [구독 해제])                    | |
| |                                                                | |
| | 대시보드 내 알림                                               | |
| | [v 활성화]  (항상 활성)                                       | |
| +---------------------------------------------------------------+ |
|                                                                    |
| +-- 알림 유형별 설정 ------------------------------------------+ |
| |                                                                | |
| |                           | Slack | 웹 푸시 | 대시보드 |       | |
| | 수동 개입 요청            | [v]   | [v]     | [v]      |       | |
| | 채증 완료                 | [ ]   | [ ]     | [v]      |       | |
| | 채증 실패                 | [v]   | [v]     | [v]      |       | |
| | 시스템 오류               | [v]   | [v]     | [v]      |       | |
| | 일일 요약                 | [v]   | [ ]     | [v]      |       | |
| +---------------------------------------------------------------+ |
|                                                                    |
| +-- 근무 시간 설정 --------------------------------------------+ |
| |                                                                | |
| | 근무 외 시간 알림 억제                                        | |
| | [v 활성화]                                                    | |
| | 근무 시간: [09:00] ~ [18:00]                                  | |
| +---------------------------------------------------------------+ |
|                                                                    |
| [저장]                                                             |
+------------------------------------------------------------------+
```

#### 컴포넌트

| 컴포넌트 | 출처 | 용도 |
|----------|------|------|
| `Card` | shadcn/ui | 섹션 래퍼 |
| `Switch` | shadcn/ui | 채널별 활성/비활성, 알림 유형별 토글 |
| `Input` | shadcn/ui | Webhook URL, 시간 입력 |
| `Button` | shadcn/ui | 테스트 전송, 구독 등록/해제, 저장 |
| `Checkbox` | shadcn/ui | 알림 유형별 채널 선택 매트릭스 |
| `toast` (Sonner) | shadcn/ui | 테스트 전송/저장 결과 |

#### 인터랙션

| 동작 | 반응 |
|------|------|
| Slack "테스트 전송" 클릭 | Slack Webhook으로 테스트 메시지 전송 → toast 결과 |
| "구독 등록" 클릭 | Service Worker 등록 → `PushManager.subscribe()` → toast |
| "구독 해제" 클릭 | `PushSubscription.unsubscribe()` → toast |
| 알림 유형 체크 변경 | 즉시 상태 반영 (저장 시 적용) |
| "저장" 클릭 | 알림 설정 저장 → toast |
