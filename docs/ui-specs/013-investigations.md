# Task 013: 채증 모니터링 페이지

## 페이지 목적

1~3단계 채증 파이프라인의 전체 작업 큐 현황을 모니터링하고, 개별 채증 세션의 실시간 진행 상황을 추적하며, 완료된 채증 결과를 스크린샷/HTML/네트워크 로그 등 다양한 형태로 상세 열람한다. CAPTCHA/OTP 수동 개입이 필요한 세션도 이 페이지 그룹에서 관리한다.

## 페이지 목록

| 라우트 | 설명 |
|--------|------|
| `/investigations` | 채증 작업 큐 대시보드 (요약 카드 + 목록 테이블) |
| `/investigations/[id]` | 개별 채증 세션 실시간 상세 모니터링 |
| `/investigations/[id]/results` | 완료된 채증 결과 상세 (탭: 스크린샷/HTML/네트워크/WHOIS/메타데이터/다운로드) |
| `/investigations/captcha-queue` | CAPTCHA/OTP 수동 개입 큐 |
| `/investigations/gallery` | 채증 결과 갤러리 뷰 (스크린샷 그리드) |

**상태별 요약 카드:**

| 카드 | 필터 | 배경색 |

| 실패 | `FAILED` | red |

**필터 옵션:**

| 필터 | 타입 | 옵션 |

| 상태 | Select | 전체 / 대기 / 진행 중 / 1단계 완료 / 2단계 완료 / 3단계 완료 / 완료 / 실패 / 취소 |
