#### CAPTCHA 하이브리드 3단계 전략 (2026 "Pause & Attach" 패턴)

불법 도박 사이트의 CAPTCHA는 100% 자동 풀이가 불가능하므로, **자동→보조→수동** 3단계 전략을 적용한다:

| 단계 | 방식 | 처리 시간 | 성공률 | 비용 |
|------|------|----------|--------|------|
| 1차 | CapSolver (AI) | 3~9초 | 95% | $0.80/1K |
| 2차 | 2Captcha (사람) | 10~60초 | 99% | $1~3/1K |
| 3차 | 대시보드 수동 풀이 | 사용자 대기 | 100% | 인건비 |

**3차 수동 개입 구현**: Playwright의 CDP(Chrome DevTools Protocol)를 활용하여 원격 브라우저 세션을 대시보드에 스트리밍하고, 사용자가 직접 CAPTCHA를 풀면 자동화가 재개되는 "Pause and Attach" 패턴 적용.

- [CapSolver Playwright 연동 가이드](https://www.capsolver.com/blog/All/how-to-integrate-playwright)

### 3.2 디지털 증거 수집 법적 요건

한국 법원에서 인정하는 디지털 증거의 3대 요건:

1. **동일성**: 원본과 사본이 동일함을 증명 (해시값 비교)
2. **무결성**: 수집 시점부터 법정 제출까지 변경되지 않음을 보증
3. **신뢰성**: 수집 과정에서 적용된 기술의 신뢰성 및 타당성 입증

#### 필수 조치사항
- 수집 즉시 **SHA-256 해시값** 생성 및 기록
- 모든 수집 과정 **타임스탬프** 기록 (UTC 기준)
- 수집 과정 **영상 촬영** 또는 상세 로그 기록
- **전문가 감정서** 작성 (수집 절차, 분석 결과 포함)
- **체인 오브 커스터디(Chain of Custody)** 관리

#### 관련 규정
- 대검찰청 예규: "디지털 증거의 수집·분석 및 관리 규정"
- 법원 행정규칙: "디지털 증거의 처리 등에 관한 규칙"
- 참고: https://www.law.go.kr/LSW/admRulLsInfoP.do?admRulSeq=2100000204289

## 4. 기술 트렌드

### 4.1 AI 기반 불법 사이트 탐지
- **URL 분류**: LightGBM, XGBoost, Random Forest 등 ML 모델로 악성 URL 분류
- **딥러닝**: LSTM, CNN, CNN+LSTM 하이브리드, ELECTRA 기반 모델 (정확도 99%)
- **LLM 활용**: RoBERTa-Large로 URL 임베딩 생성, 수동 피처 엔지니어링 불필요
- **웹 콘텐츠 분석**: HTML 구조, 이미지, 텍스트, Script 기반 유사도 산출 (Gamble Tracker 방식)

### 4.2 안티봇 기술과 우회 방법
- **안티봇 기술**: Cloudflare Turnstile, Akamai Bot Manager, DataDome
- **우회 기술**: puppeteer-extra-plugin-stealth, selenium-stealth
  - navigator.webdriver 플래그 제거
  - Chrome 런타임 속성 정규화
  - 권한 API 정규화
  - WebGL 메타데이터 패치
  - 폰트 렌더링 이상 수정

### 4.3 브라우저 핑거프린팅 동향
- **수집 데이터**: 화면 해상도, 시스템 폰트, GPU 타이밍, TLS 암호화 순서 등 수십 개 데이터 포인트
- **대응**: 안티디텍트 브라우저로 프로필별 핑거프린트 격리
- **트렌드**: Canvas, WebGL, Audio Context 핑거프린팅이 주류
- **도전**: TLS 핑거프린팅(JA3/JA4)은 브라우저 수준에서 우회 어려움

### 4.4 블록체인 기반 증거 보존 (신규 트렌드)
- 디지털 증거를 블록체인에 기록하여 무결성 보장

```
1위: SMS OTP 인증 ← 핵심 해결 과제
2위: 도메인 호핑   ← 실시간 추적 필요
3위: 팝업/광고     ← 자동 닫기 필요
4위: Cloudflare    ← 자동 통과 가능

```

### 5.4 비용 영향

CAPTCHA 관련 비용이 기존 예상($50~200/월)에서 대폭 하향(~$2/월). 실제 비용 중심은:
- **SMS 인증**: $100~300/월 (가상번호 발급)
- **프록시**: $100~500/월 (레지덴셜 IP)
- **CAPTCHA**: ~$2/월 (출현 빈도 낮음)

## 6. 결론 및 시사점

1. **차별화 기회**: 기존 솔루션은 "탐지"에만 집중하며, 탐지부터 법적 증거 수집까지 자동화하는 통합 솔루션은 부재
2. **기술 성숙도**: Playwright + 스텔스 플러그인 + CAPTCHA 솔버 + SMS 서비스를 조합하면 기술적으로 구현 가능 — **실제 사이트 테스트로 검증 완료**
3. **법적 요건 충족**: SHA-256 해시, RFC 3161 타임스탬프, Chain of Custody 관리로 디지털 증거의 법적 유효성 확보 가능
4. **정부 수요 존재**: 사행산업통합감독위원회, 경찰청 등에서 AI 기반 불법 도박 감시 시스템 예산 확보 및 도입 추진 중
5. **리스크 관리 필요**: ~~안티봇 기술~~ → **SMS OTP 가상번호 차단**과 **도메인 호핑**이 실제 최대 리스크 (2026-03-14 테스트 확인)
