### 프론트엔드

- **Next.js 15** (App Router) - React 풀스택 프레임워크
- **TypeScript 5.6+** - 타입 안전성
- **React 19** - UI 라이브러리
- **TailwindCSS v4** - 유틸리티 CSS
- **shadcn/ui** - React 컴포넌트
- **React Hook Form 7.x** + **Zod** - 폼 + 검증

### 백엔드 & 데이터베이스

- **PostgreSQL** - 관계형 DB

### 배포

- **Vercel** - Next.js 최적화 배포
